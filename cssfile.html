What is CSS?

CSS stands for Cascading Style Sheets. It controls the look and feel of web pages. With CSS, we can change colours, fonts, spacing, and the layout of elements on the page.

HTML is for structure. CSS is for presentation.

CSS Features

  • Controls colours and fonts
  • Manages spacing using margin and padding
  • Helps create layouts with box model and positioning
  • Makes navigation bars and menus
  • Can create hover effects using pseudo classes

Example: CSS Properties Table

Property Use
color Sets the text colour
background-color Sets the background colour
margin Creates space outside the border
padding Creates space inside the border
border Adds a border around the element
